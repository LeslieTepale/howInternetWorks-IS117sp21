Encryption Public Keys

The internet is a public system in that many servers and computers communicate to share information, despite the fact that some of that is private. This can include credit card information, bank statements or passwords.


Encryption

Encryption jumbles information so that the original message is scrambled into something different, to hide the information that can be private. Decryption unscrambles that message to get the message.
A key is a password that is only known by the sender and the receiver to decrypt a message. At times it can be difficult to encrypt a message securely as they are easily solvable and therefore easily hackable. Therefore, keys that include more digits and have much longer variation are much more secure as it would take a human many centuries to decrypt a message.

Symmetric & Asymmetric Encryption

Symmetric encryption is when the sender and receiver use the same key to both encrypt and decrypt a message. Though it's better for humans to come up with keys this way, computers are much more vulnerable because they are less private.
Computers use different keys for encryption and decryption for the message to be better protected. Public keys can be used to encrypt messages but private keys are those that can decrypt messages and only a computer has access to it.
